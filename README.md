# MacOS Icon Repo

This repository is maintained to provide up-to-date, https accessible icons for use in macOS deployments. 
Common tools that I use to leverage this repository include: 
 - [Jamf Setup Manager](https://github.com/jamf/Setup-Manager)
 - [Setup Your Mac](https://github.com/setup-your-mac/Setup-Your-Mac) / [Setup Your Mac Lite](https://github.com/setup-your-mac/SYM-Lite)
 - [SwiftDialog](https://github.com/swiftDialog/swiftDialog)

Currently only **png** versions are being uploaded.
If there are any icons you would like uploaded or you would like other file types included, please submit your request as a discussion thread. 

## Usage 
Ensure you use the public endpoint to reference any icons you wish to use in your deployment tools: 
```https://raw.githubusercontent.com/litlabsau/macos-icons/refs/heads/main/png/{IconName}.png```
Alternatively, the repo can be cloned if you prefer the icons to be available locally.
